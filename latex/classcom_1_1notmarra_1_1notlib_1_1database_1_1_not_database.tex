\doxysubsection{Detailed Description}
An abstract base class for database operations with configurable settings. 

\doxylink{classcom_1_1notmarra_1_1notlib_1_1database_1_1_not_database}{Not\+Database} provides a foundation for database interactions with built-\/in connection pooling via Hikari\+CP. It supports common SQL operations including select, insert, update, and delete operations through a fluent SQL builder interface.

The class manages database connections and provides methods for\+:
\begin{DoxyItemize}
\item Connection management via Hikari\+CP
\item Table creation and management
\item SQL query execution
\item Data type conversion
\item Configuration handling
\end{DoxyItemize}

Implementations must provide database-\/specific logic for connecting, creating tables, and inserting data.
