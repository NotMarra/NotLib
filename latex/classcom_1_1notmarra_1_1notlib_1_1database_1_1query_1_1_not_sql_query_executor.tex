\doxysubsection{Detailed Description}
Provides methods to execute SQL queries on a \doxylink{classcom_1_1notmarra_1_1notlib_1_1database_1_1_not_database}{Not\+Database} instance. 

The \doxylink{classcom_1_1notmarra_1_1notlib_1_1database_1_1query_1_1_not_sql_query_executor}{Not\+Sql\+Query\+Executor} class provides a simplified interface for executing SQL queries against a \doxylink{classcom_1_1notmarra_1_1notlib_1_1database_1_1_not_database}{Not\+Database}. It supports operations such as selecting records (returning single results or lists), updating data, checking existence of records, and counting results.
