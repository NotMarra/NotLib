\doxysubsection{Detailed Description}
Abstract SQLite implementation of the \doxylink{classcom_1_1notmarra_1_1notlib_1_1database_1_1_not_database}{Not\+Database} class. 

\doxylink{classcom_1_1notmarra_1_1notlib_1_1database_1_1source_1_1_not_s_q_lite}{Not\+SQLite} provides SQLite database functionality using Hikari\+CP connection pooling. This class handles the specifics of connecting to an SQLite database file and creating SQL statements compatible with SQLite syntax, such as table creation and row insertion.

The class establishes a connection to an SQLite database specified in the plugin configuration. It configures Hikari\+CP connection pool with SQLite-\/appropriate settings and provides methods for database operations like table creation and data insertion.

SQLite-\/specific features\+:
\begin{DoxyItemize}
\item Uses file-\/based database storage
\item Implements AUTOINCREMENT instead of standard AUTO\+\_\+\+INCREMENT
\item Creates SQLite-\/compatible table schemas
\end{DoxyItemize}

\begin{DoxyNote}{Note}
When working with SQLite, be aware of its limitations compared to other RDBMS, including limited concurrent access and transaction isolation. 
\end{DoxyNote}
