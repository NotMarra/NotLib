\doxysubsection{Detailed Description}
A fluent SQL query builder for constructing SQL queries of various types. 

\doxylink{classcom_1_1notmarra_1_1notlib_1_1database_1_1query_1_1_not_sql_builder}{Not\+Sql\+Builder} provides a convenient and type-\/safe way to build SQL queries for\+:
\begin{DoxyItemize}
\item SELECT\+: retrieving data from tables
\item INSERT\+: adding new records to tables
\item UPDATE\+: modifying existing records
\item DELETE\+: removing records from tables
\end{DoxyItemize}

This class follows the builder pattern, allowing method chaining for a clean API.

Example usage\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{comment}{//\ Create\ a\ SELECT\ query}}
\DoxyCodeLine{String\ query\ =\ NotSqlBuilder.select(\textcolor{stringliteral}{"{}users"{}})}
\DoxyCodeLine{\ \ \ \ .columns(Arrays.asList(\textcolor{stringliteral}{"{}id"{}},\ \textcolor{stringliteral}{"{}name"{}},\ \textcolor{stringliteral}{"{}email"{}}))}
\DoxyCodeLine{\ \ \ \ .whereEquals(\textcolor{stringliteral}{"{}active"{}},\ 1)}
\DoxyCodeLine{\ \ \ \ .orderBy(\textcolor{stringliteral}{"{}name"{}})}
\DoxyCodeLine{\ \ \ \ .limit(10)}
\DoxyCodeLine{\ \ \ \ .build();}

\end{DoxyCode}


The builder handles proper query formatting and parameter binding to help prevent SQL injection.
