\doxysubsection{Detailed Description}
A builder class for constructing SQL WHERE clauses in a fluent API style. 

This class provides a convenient way to build complex SQL WHERE conditions with proper parameter handling to prevent SQL injection. It supports various comparison operators, logical AND/\+OR combinations, and special conditions like IN, LIKE, and NULL checks. \begin{Desc}
