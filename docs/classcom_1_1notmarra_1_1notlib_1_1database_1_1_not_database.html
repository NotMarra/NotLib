Detailed Description

An abstract base class for database operations with configurable settings.

NotDatabase provides a foundation for database interactions with built-in connection pooling via HikariCP. It supports common SQL operations including select, insert, update, and delete operations through a fluent SQL builder interface.

The class manages database connections and provides methods for:

  • Connection management via HikariCP
  • Table creation and management
  • SQL query execution
  • Data type conversion
  • Configuration handling

Implementations must provide database-specific logic for connecting, creating tables, and inserting data.
