Detailed Description

Provides methods to execute SQL queries on a NotDatabase instance.

The NotSqlQueryExecutor class provides a simplified interface for executing SQL queries against a NotDatabase. It supports operations such as selecting records (returning single results or lists), updating data, checking existence of records, and counting results.
